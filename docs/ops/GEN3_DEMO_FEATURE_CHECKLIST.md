# Gen3 演示版功能清单（对外）

## 产品定位
- 面向复杂合同（优先 FIDIC 工程合同）的 AI + 人审协同审阅系统。

## 已可演示核心能力
1. 领域化审阅
- 支持 FIDIC 领域审阅流程（条款级检查、风险识别、修改建议）。

2. 智能分析与建议生成
- 自动解析合同条款并逐条审阅。
- 输出风险点与可执行文本修改建议（diff）。

3. 审批闭环（Human-in-the-loop）
- 支持逐条批准/拒绝与批量处理。
- 全部待审批项完成决策后才允许恢复流程。
- 支持“拒绝后继续生成下一批建议”。

4. 可解释审阅界面
- 字符级差异高亮（内联 diff）。
- 条款全文上下文查看。
- 建议文本可编辑后再批准。
- 处理进度、中断说明、按条款分组、决策历史。

5. 结果沉淀与导出
- 审阅完成后生成结构化总结。
- 支持导出 DOCX 红线（基于已批准建议，含用户编辑文本）。

6. 平台扩展能力
- 领域插件机制（可扩展 SHA/SPA 等交易文件）。
- 支持本地 Skills 与远程工作流（Refly）双后端策略。

## 当前可对外表述
- “支持工程合同（FIDIC）条款级智能审阅，提供风险识别、修改建议、人审审批和 DOCX 红线导出的一体化流程。”
- “已具备领域插件化扩展底座，可逐步覆盖 SHA/SPA 等复杂交易文件。”

## 非阻塞增强项（后续路线）
- Refly 远程 Skill 深化。
- RAG/向量检索能力深化。
- Legacy 路径完全退役（工程清理）。
